# RISC-V Core Security Advisories

This repository hosts public-facing security advisory packages for several open-source RISC-V cores.

Repository layout:

- `nutshell/`: public advisory packages for OSCPU NutShell
- `rocket/`: public advisory packages for Rocket Chip related findings
- `cva6/`: public advisory packages for CVA6 related findings
- `xiangshan/`: public advisory packages for XiangShan related findings

Each advisory directory may include:

- `README.md`: package overview and public reference summary
- `VULNERABILITY_REPORT.md` / `.pdf`: the main public report
- `poc/`: public proof-of-concept artifacts and evidence
- `packages/*.zip`: downloadable copies of the public package contents
